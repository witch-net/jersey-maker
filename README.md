# Jersey Maker

## Configuration

```json
{
    "address": "127.0.0.1",
    "port": 8080,
    "databaseURL": ""
}
```

### `address`

- Type: `string`
- Default: `"127.0.0.1"`

Specify the server’s listening address.

### `port`

- Type: `number`
- Default: `8080`

Specify the server’s listening port.

### `databaseURL`

- Type: `string`
- Default: **required**

Specify the URL of a MongoDB server.
